Nova - это открытый видеоплеер, разработанный для планшетов, телефонов и устройств Android TV доступных по адресу https://github.com/nova-video-player/aos-AVP

Универсальный проигрыватель:
- Воспроизведение видео с вашего компьютера, сервера (FTP, SFTP, WebDAV), NAS (SMB, UPnP)
- Воспроизведение видео с внешнего USB-накопителя
- Видео из всех источников, интегрированных в единую мультимедийную коллекцию
- Автоматический онлайн поиск описаний фильмов и телешоу с постерами и фонами
- Интегрированная загрузка субтитров

Лучший видеоплеер:
- Аппаратное ускорение декодирования видео для большинства устройств и видеоформатов
- Поддержка нескольких звуковых дорожек и мульти-субтитров
- Поддерживаемые форматы файлов: MKV, MP4, AVI, WMV, FLV и т. д.
- Поддерживаемые типы файлов субтитров: SRT, SUB, ASS, SMI и т. д.

ТВ дружественный:
- Специальный интерфейс для пользователей Android TV
- Переход через AC3 / DTS (HDMI или S / PDIF) на поддерживаемых часах Ardware: Nexus Player, NVidia SHIELD TV, медиаприставках на базе Rockchip и AmLogic
- Поддержка 3D с параллельным и верхним нижним воспроизведением форматов для 3D-телевизоров
- Режим Audio Boost для увеличения уровня громкости
- Ночной режим для динамической регулировки уровня громкости

Просматривайте по своему усмотрению:
- Мгновенный доступ к недавно добавленным и недавно воспроизведенным видео
- Просмотр фильмов по имени, жанру, году, продолжительности, рейтинг
- Просмотр ТВ-шоу по сезонам
- Поддерживается просмотр папок

И даже больше:
- Продолжить воспроизведение видео  из сети на других устройствах
- Обработка метаданных NFO для описаний и постеров
- Запланированное повторное сканирование содержимого вашей сети (только для интерфейса Leanback)
- Частный режим: временно отключить запись истории воспроизведения
- Ручная настройка синхронизации субтитров
- Ручная настройка синхронизации аудио / видео
- Отслеживайте свой коллекцию и то что вы хотите посмотреть через Trakt

Если у вас есть проблема или вопрос о приложении, пожалуйста, проверьте нашу группу поддержки Reddit по этому адресу: https://www.reddit.com/r/NovaVideoPlayer

Если у вас возникли проблемы с аппаратным декодированием видео, вы можете принудительно выполнить программное декодирование в настройках приложения.

Вы можете внести свой вклад в перевод приложения по адресу https://crowdin.com/project/nova-video-player
